Application and Placement Tracker

Workflow 1
Application Logging and Updating

This would include basic CRUD operations. Enable users to organize all applications in one place with status columns, role categories, and deadline awareness.

1.  Application Dashboard: The user logs in and is directed to the main tracker dashboard.
2.  Adding a New Entry: The user clicks the '+ Add Application' button.
3.  Data Input: A form appears requiring key details:
Company Name
Role (Links to Workflow 2 for the type of resume to be uploaded)
Status
Salary
Job Type 
Location
URL
Deadline
Application Date
The user clicks 'Save'. The form closes and the new entry card is immediately added to the dashboard view in the appropriate status column.
4.  Updating or deleting an Entry: The user would click on the card in order to update the status or any other details. They can also delete the entry.


Workflow 2 
Resume Library

Help users manage multiple resumes for different roles and track which resume was used for each application 

1. Navigate to Resume Library
    The user clicks ‘Resume Library’ from the sidebar/tab.
2. Add/Upload Resume
    The user clicks ‘+ Add Resume’ and uploads or saves a resume entry.
    The user then chooses its category/track.
3.Resume List View
    The system shows a list of saved resumes with:
        - Resume name
        - Category/track
        - Applications it has been used for
        - Last updated date
        - Buttons: View / Delete
4. Link Resume to Application (from Add Application)
    The user goes to ‘Add Application’ form and selects “Resume Used” from a dropdown of saved resumes.
5. Search resumes 
6. Resume Usage Tracking
    Confirmation ‘Resume linked successfully.’

Workflow 3 

Track the interview process in detail (rounds, dates, notes, follow-up reminders) so users don’t lose information. Also visualize the success rate of interviews through percentage calculations through Offer or Rejection after Interview, data analytics of reaching the interview stage, etc.
1. Navigate to Interview Tracker
The user clicks “Interview Tracker” from the sidebar.
2. Interview Dashboard View
The system displays only applications in Interview stage, with cards showing:
- Company, role
- Interview date/time (if added)
- Round
- Status 
3. Interview Details
The user clicks an interview-stage application “Add Interview Details”.
They fill:
- Interview date & time
- Round type
- Mode (Onsite/Online) + link (optional)
- Interviewer (optional)
- Preparation checklist (optional)
The user saves.
4. Preparation Notes
- The user adds notes like:
- Topics to revise
- Questions asked
- Company-specific prep
Saves.
5. Post-Interview Update (optional)
After the interview, the user updates the status.
6. Follow-up Reminders (optional)
The user sets a follow up date.
The tracker highlights pending follow-ups.
7. Confirmation
