Resume Library

Help users manage multiple resumes for different roles and track which resume was used for each application 

1. Navigate to Resume Library
    The user clicks ‘Resume Library’ from the sidebar/tab.
2. Add/Upload Resume
    The user clicks ‘+ Add Resume’ and uploads or saves a resume entry.
    The user then chooses its category/track.
3.Resume List View
    The system shows a list of saved resumes with:
        - Resume name
        - Category/track
        - Applications it has been used for
        - Last updated date
        - Buttons: View / Delete
4. Link Resume to Application (from Add Application)
    The user goes to ‘Add Application’ form and selects “Resume Used” from a dropdown of saved resumes.
5. Search resumes 
6. Resume Usage Tracking
    Confirmation ‘Resume linked successfully.’
